# WREN Admin UI (Legacy)

The original React-based admin interface for [WREN](https://wren.aemwip.com). Served at `/oldadmin`.

**Note:** This has been superseded by [adminui2](https://github.com/usewren/adminui2), which is a vanilla JS rewrite with no build dependencies. This repo is kept for reference.
